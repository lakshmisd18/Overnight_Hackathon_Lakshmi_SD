# Overnight_Hackathon_Lakshmi_SD
A simple system that protects digital content by generating a unique hash for every uploaded image. The hash is stored securely, and users can verify later whether an image is original or copied by comparing hashes. This helps creators prove ownership and prevent copyright misuse.
